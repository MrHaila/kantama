# Opas

Web frontend for Kantama. Interactive Vue 3 + D3.js visualization of transit travel times.

## Development

```bash
pnpm dev      # Start dev server
pnpm build    # Production build
pnpm test     # Run E2E tests
pnpm lint     # Lint code
```

## Data

Reads from `public/data/` (generated by Varikko):
- `zones.json` - Zone metadata and reachability scores
- `routes/*.msgpack` - Per-zone route data (loaded on demand)
- `background_map.json` - Static map layers

See `AGENTS.md` for architecture details.
